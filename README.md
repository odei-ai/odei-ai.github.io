# odei.ai

**The public research surface for ODEI.**

This repository contains the static site deployed at [odei.ai](https://odei.ai).

Its role is not to act as a sales page.
Its role is to communicate the thesis behind ODEI:

- human agency needs memory, policy, and continuity
- personal AI requires a user-owned world model
- safety must live in architecture, not only in prompts
- real symbiosis needs auditability, not chat history

## Surface Role

`odei.ai` is the **research and narrative layer** of the ODEI stack.

It sits alongside:

- [api.odei.ai](https://api.odei.ai) for production APIs and docs
- [research](https://github.com/odei-ai/research) for papers
- [memory](https://github.com/odei-ai/memory) for memory architecture

## Tech

- static HTML and CSS
- GitHub Pages deployment
- minimal build complexity

## Development

Open the site locally:

```bash
open index.html
```

Edit, refresh, and deploy on push to `main`.
