# odei.ai — ODEI Landing Page

[![Deploy Status](https://img.shields.io/github/actions/workflow/status/odei-ai/odei-ai.github.io/pages/pages-build-deployment?label=deploy&style=flat-square)](https://github.com/odei-ai/odei-ai.github.io/actions)
[![Website](https://img.shields.io/website?url=https%3A%2F%2Fodei.ai&style=flat-square&label=odei.ai)](https://odei.ai)

Landing page for **ODEI** — Human-AI Symbiosis. Served via GitHub Pages at [odei.ai](https://odei.ai).

ODEI (Operational Decision and Execution Intelligence) is a graph-native world model with constitutional safety guardrails for autonomous AI agents. This repository contains the public-facing landing page.

## Tech

- Static HTML + CSS
- Responsive design
- No build step required

## Development

Open `index.html` in your browser:

```bash
open index.html
```

No dependencies to install, no build process. Edit and refresh.

## Deployment

Automatic via **GitHub Pages** on push to `main`. Changes go live within minutes.

Custom domain configured to [odei.ai](https://odei.ai) via CNAME.
