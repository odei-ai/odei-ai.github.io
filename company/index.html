Company Overview

ODEI builds personal AI operating systems.

ODEI is a software product company building governed AI infrastructure for individuals and AI-native operators. The product combines persistent memory, a graph-based world model, constitutional guardrails, and auditable execution so an AI system can act with continuity, user control, and clear provenance.

What ODEI does

ODEI provides a constitutional world model for AI agents. Instead of relying on isolated prompts and session-bound chat history, ODEI keeps user context in a persistent graph, routes actions through policy checks, and records why the system acted. The result is an AI control layer that can compound knowledge over time while remaining inspectable and user-governed.

  • Persistent memory across sessions with user-owned context and provenance.
  • Graph-native world model for goals, tasks, entities, evidence, and operating context.
  • Constitutional guardrails applied before AI actions are executed.
  • Public API and documentation at api.odei.ai.
ODEI is a product company. It is not a consultancy, dev shop, agency, personal blog, or content site.
